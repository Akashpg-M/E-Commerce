## E-Commerce Application Overview

###  **Project Summary**
Built a full-stack **E-Commerce application** with **role-based access control (RBAC)** for efficient user and admin management. The app supports secure authentication, optimized data retrieval, and seamless shopping experiences.

### 🚀 **Key Features**
- **Role-Based Access**
  - 🔹 **Admin**: Manages inventory (add/update stock, product details).
  - 🔹 **User**: Can browse products, add to cart, and proceed to checkout.

- **Secure Authentication**
  - Implemented **JWT authentication** using **access & refresh tokens** for enhanced security.

- **Performance Optimization**
  - **Redis caching** speeds up **token verification** and **product retrieval**, improving app responsiveness.

### 💻 **Tech Stack**
- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Caching & Performance**: Redis
- **State Management**: Zustand
